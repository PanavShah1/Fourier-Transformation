Fourier Transformation

In physics, engineering and mathematics, the Fourier transform (FT) is an integral transform that converts a function into a form that describes the frequencies present in the original function. The output of the transform is a complex-valued function of frequency.

This program selects discrete points on the graph of the function and approximates the corresponding Fourier transformation

How to use :

  • Enter the range and precision
  • Enter the equation (use brackets for every function eg. sin(x), ln(x), sqrt(x))
  • Press "Go"
